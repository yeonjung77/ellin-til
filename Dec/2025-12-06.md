# router & controller
    - router : 주소/경로 등로
        - URL 경로 등록
        - GET/POST 같은 HTTP Method 지정
        - 들어온 요청을 어떤 controller 함수로 보낼지 연결
    - controller : 기능 구현
        - 비즈니스 로직 실행(DB 조회, 데이터 가공, 검증 로직, 예외 처리)

# Homebrew 
    - Mac에서 프로그램을 쉽게 설치/삭제해주는 패키지 관리자

# PostgreSQL
    - DB 시스템(오픈소스 관계형 데이터베이스)
    - 데이터를 안전하게 저장, 검색, 수정, 삭제할 수 있게 해주는 프로그램
    - brew install postgresql : PostgreSQL 설치
    - brew services start postgresql : 실행
    - psql postgre : 데이터베이스 접속
