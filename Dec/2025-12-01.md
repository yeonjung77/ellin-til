# num_workers란?

- DataLoader가 데이터를 불러올 때 사용할 CPU 작업자(worker) 수를 의미
- 숫자가 클수록 데이터 로딩을 병렬로 처리해서 학습 속도가 빨라지고, 기본값은 0(병렬 없음)

# breakpoint()

- 코드 실행을 특정 줄에서 멈추게 하는 디버깅 도구
- 멈춘 시점에서 변수 값, 함수 흐름, 에러 원인 등을 직접 확인 가능


# FAISS와 BM25

- FAISS : 의미 기반 검색(semantic search)
    - 문서의 의미를 벡터로 바꿔서 검색함
    - 단어가 다르더라도 의미가 비슷하면 잘 찾음
    - 장점 : 의미 기반 검색에 강함 / 단점 : 키워드가 정확해도 의미 벡터가 못잡는 경우 있음

- BM25 : 전통적인 키워드 기반 검색(keyword search)
    - 검색어에 정확히 등장하는 단어가 많이 포함된 문서에 높은 점수를 줌
    - 장점 : 키워드 정확도가 높다 / 단점 : 단어가 다르면 의미가 같아도 못찾음

- hybrid search :
    - FAISS만 쓰면 비슷한 의미를 가진 문장을 잘 찾지만 숫자, 고유명사에는 약함
    - BM25만 쓰면 같은 키워드를 잘 찾지만 의미가 같은데 다른 단어로 쓰면 놓침
