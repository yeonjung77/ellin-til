# 웹 프레임워크
    - 웹사이트나 서비스를 만드는 데 쓰는 파이썬 도구
    - Django : 풀세트, 기능이 많음, 쇼핑몰/커뮤니티에 적합
    - Flask : 가벼움, 자유도 높음, 간단한 웹/ 실험용 API에 적합
    - FastAPI : 빠름, 데이터를 주고받는 API 개발에 최적화, AI모델API서버/데이터 서비스에 적합
    - uvicorn : FastAPI를 실행시켜주는 서버 프로그램(코드를 실행해서 인터넷 요청을 받아 처리하는 서버)

# pip freeze 
     - 현재 파이썬 환경에 설치된 모든 라이브러리 목록을 정확한 버전까지 보여주는 명령어
     - pip freeze > requirements.txt : 현재 설치된 라이브러리 목록을 requirements 파일에 기록 (shift + .)
     - pip install -r requirements.txt : 저장된 환경을 그대로 재현
