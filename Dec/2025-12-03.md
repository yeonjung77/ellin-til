# google ai studio
    - 브라우저에서 Gemini 모델을 바로 테스트하고, 프롬프트·API 코드 스니펫을 자동으로 생성해주는 플랫폼
    - 모델 Playground + API 콘솔 역할을 해서, 이미지/텍스트/코드 모두 실험한 뒤 바로 개발에 적용 가능

# github gemini cookbook
    - Gemini 모델을 활용한 예제 코드 모음집으로, 텍스트·이미지·멀티모달·RAG 등 실전 구현 레시피를 제공

# 이미지 임베딩 -> 유사 이미지 검색 과정
    - 이미지 입력 → 사전학습 모델(예: CLIP/Gemini Vision Encoder)로 벡터(임베딩)로 변환
    - 전체 이미지 벡터들을 FAISS 같은 벡터 DB에 저장해두고, 입력 이미지 벡터와 가장 가까운 벡터를 찾아 유사 이미지를 반환
